# 发票识别工具

一款用于识别和整理发票的 GUI 工具。

## 功能特性

- **自动识别**：从 PDF 文件中自动提取发票信息
- **支持类型**：
  - 火车票（自动识别站点、日期、金额）
  - 滴滴电子发票 + 行程报销单（按金额配对）
  - 餐饮发票
  - 通行发票
  - 车辆租赁
  - 话费
  - 机票
- **整理功能**：按火车票日期区间整理发票
- **统计报告**：自动生成发票统计报告

## 使用方法

### 1. 运行程序

双击 `启动GUI.bat` 或在命令行运行：
```
python invoice_gui.py
```

### 2. 识别发票

1. 点击"开始识别"
2. 程序会自动从 PDF 中提取信息

### 3. 整理发票

1. 点击"按火车票区间整理发票"
2. 选择开始和结束的火车票
3. 确认整理

## 依赖

- Python 3.8+
- pdfplumber

安装依赖：
```
pip install pdfplumber
```
