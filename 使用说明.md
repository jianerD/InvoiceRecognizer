# 发票识别工具 - 使用说明

## 方式一：直接运行（推荐）

1. 确保已安装 Python 3.8+
2. 安装依赖：
   ```
   pip install pdfplumber
   ```
3. 运行程序：
   ```
   python invoice_gui.py
   ```

---

## 方式二：打包成 exe（需要管理员权限）

1. 以**管理员身份**运行 PowerShell
2. 进入程序目录
3. 运行：
   ```
   pip install pyinstaller
   pyinstaller --onefile --noconsole invoice_gui.py
   ```
4. 打包后的文件在 `dist\InvoiceTool.exe`

---

## 方式三：便携版（推荐发给别人）

打包一个包含 Python 环境的便携版：

1. 下载 Python 便携版：https://www.python.org/ftp/python/
2. 解压到程序目录，重命名为 `python_portable`
3. 使用便携版 Python 安装依赖并打包
